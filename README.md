# 🌱 Crop Recommendation System (Mini Project)

## 📌 Overview
This is a simple, web-based mini project designed to recommend the most suitable crop for cultivation based on specific soil and weather conditions. Users can enter environmental data into a web form, and the Python backend processes it to predict the best crop to grow.

## 🛠️ Tech Stack
* **Frontend:** HTML, CSS, JavaScript (for the user interface and web form)
* **Backend / Logic:** Python (processes the data and handles the recommendation logic)

## 📊 Dataset Details
The recommendation model relies on the following key input features provided by the user:
* **N** - Nitrogen content ratio in the soil
* **P** - Phosphorous content ratio in the soil
* **K** - Potassium content ratio in the soil
* **temperature** - Temperature in degrees Celsius
* **humidity** - Relative humidity percentage
* **ph** - pH value of the soil
* **rainfall** - Rainfall in millimeters
